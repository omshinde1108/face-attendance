📋 Setup Guide
  1. Go to script.google.com → New project
  2. Paste code from google-apps-script.js
  3. Replace YOUR_SHEET_ID with your Google Sheet ID
  4. Deploy → Web App → Anyone → Deploy
  5. Copy Web App URL → paste above in Settings
  6. Upload all files to GitHub → enable GitHub Pages
  7. Your live URL: https://username.github.io/face-attendance
